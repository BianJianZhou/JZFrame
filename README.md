# JZFrame  
帮助更多开发者快速搭建APP，并尽我所能提供生涯中问题/想法等的解决方案  
## 依赖方式  
Add it in your root build.gradle at the end of repositories:
`<allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}>`  

Add the dependency
`<dependencies {
	        implementation 'com.github.Mr-Bian:JZFrame:Tag'
	}>`
